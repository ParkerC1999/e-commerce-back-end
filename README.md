# e-commerce-back-end

## Purpose:
The purpose of this application is to organize objects in a database. New objects can be created and all objects can be searched (all or by id), updated, or deleted.

## Built With:
* Javascript
* Node.js
    * Express
    * MySQL
    * Sequelize
